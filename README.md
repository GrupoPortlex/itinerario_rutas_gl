# Itinerario de rutas — ESMAS-ES+ — Galego

## Descrición

**Itinerario de rutas - ESMAS-ES+ - Galego** é unha guía interactiva dun sistema de clasificación semántica. Este recurso organiza rutas ontoló­xicas en distintos mundos conceptuais, abrangue unha ampla variedade de temas. Cada ruta vai acompañada de numerosos exemplos léxicos para ilustrar o seu significado, convertendo esta ferramenta nun recurso completo para a exploración do léxico.
