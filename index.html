Descrición

Itinerario de rutas - ESMAS-ES+ é unha guía interactiva dun sistema de clasificación semántica. Este recurso organiza rutas ontolóxicas en distintos mundos conceptuais, abarcando unha ampla variedade de temas. Cada ruta está acompañada de numerosos exemplos léxicos para ilustrar o seu significado, converténdoo nunha ferramenta completa para a exploración do léxico.

Como usar esta ferramenta?

Para explorar o contido deste itinerario, podes navegar a través dos distintos mundos conceptuais, como, por exemplo, "Animado" ou "Material". Se buscas unha ruta específica, podes utilizar o filtro de busca para atopala facilmente.

Cada ruta preséntase cunha estrutura xerárquica que te axuda a entender a súa clasificación. Ao seleccionar unha, atoparás unha lista de palabras que pertencen a esa categoría, servindo como exemplos ilustrativos.
